## 🚀 How to Run

1. Open **app/ArogyaBot_Final.html** in **Google Chrome**
2. App loads instantly — no installation needed!
3. AI is pre-configured and ready to use
4. Allow location permission for hospital finder

---

## 🛠️ Tech Stack

| Layer | Technology |
|---|---|
| Frontend | HTML5 + CSS3 + Vanilla JavaScript |
| AI Engine | Groq API — Llama 3.3 70B (Free!) |
| Database | Supabase (PostgreSQL) — Cloud sync |
| Voice Input | Web Speech API (browser built-in) |
| Maps | OpenStreetMap + Nominatim (Free) |
| Directions | Google Maps URL API |
| Storage | localStorage + Supabase |
| Fonts | Sora + Noto Sans Devanagari |

---

## ✨ Features

| Feature | Description |
|---|---|
| 💬 AI Symptom Checker | Groq Llama 3.3 — Hindi & English |
| ⚠️ Severity Detection | LOW / MEDIUM / HIGH with color badges |
| 🎤 Voice Input | Speak symptoms in Hindi or English |
| 🌐 Language Toggle | Switch EN/HI instantly |
| 🏥 Hospital Finder | GPS-based — finds nearest PHC/hospital |
| 🔔 Medicine Reminders | Add, view, delete with WhatsApp share |
| ☁️ Cloud Database | All data saved to Supabase in real time |
